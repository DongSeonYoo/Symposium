# Symposium — 위기 대응 모드 (블랙스완 프로토콜)

## 개요

이란 사태(2026.02.28)와 같은 지정학적 충격, 금융위기, 자연재해 등
시장 외부 충격 발생 시 시스템이 자동으로 위기 모드로 전환.

## 자동 감지 트리거

아래 조건 중 하나라도 충족 시 위기 모드 진입:

| 트리거 | 임계값 | 데이터 소스 |
|--------|--------|------------|
| VIX 급등 | > 40 또는 단기 +30% | Yahoo Finance |
| 유가 급등 | 단기 +10% 이상 | Yahoo Finance |
| 달러인덱스 급변 | 단기 ±3% 이상 | Yahoo Finance |
| 뉴스 감성 점수 | 전쟁/위기 키워드 임계값 초과 | symposium-news |
| KOSPI 서킷브레이커 | 거래소 발동 감지 | KIS API |
| 사용자 수동 전환 | 대시보드 직접 활성화 | Dashboard |

## 일반 모드 vs 위기 모드

| 항목 | 일반 모드 | 위기 모드 |
|------|----------|----------|
| 분석 주기 | 매일 08:50 | 30분 간격 실시간 |
| 신규 매수 판단 | 정상 생성 | **전면 보류** |
| 기존 포지션 | 정상 유지 | 손절가 하향 조정 제안 |
| Confirm 타이머 | 30분 자동 만료 | **타이머 없음**, 수동 해제 |
| 페르소나 토론 주제 | 종목별 매매 판단 | 위기 대응 전략 특화 |
| 달리오/소로스 가중치 | 일반 | **자동 상향** |
| 알림 | 대시보드 뱃지 | 긴급 배너 + 이메일 |
| 현금 비중 제안 | 없음 | 자동 제시 |

## 위기 모드 토론 주제 템플릿

```
현재 [이벤트명] 상황이 한국 주식시장에 미치는 영향과
섹터별 대응 전략을 논의하시오.

거시경제 컨텍스트: [MacroContext 주입]
현재 포트폴리오: [보유 종목 및 평가손익]
```

## 위기 시 섹터 영향 분류 (이란 사태 예시)

| 섹터 | 영향 | 대표 종목 |
|------|------|---------|

## 대시보드 위기 모드 UI

- 상단 빨간 배너: `⚠️ 위기 모드 활성 — 신규 매수 보류 중`
- 현재 포지션 전체 평가손익 실시간 표시
- 위기 원인 뉴스 요약 카드 (페르소나 토론 결과 포함)
- 섹터별 영향도 히트맵 (수혜/피해 섹터 시각화)
- `위기 모드 해제` 버튼 (수동 or 트리거 정상화 시 자동 제안)

## 위기 이후 복귀

1. 트리거 지표 정상화 감지 → 대시보드에 복귀 제안
2. 사용자 확인 후 일반 모드 전환
3. 위기 기간 중 판단/대응 전체 DB 보관 → 전략 리뷰 자료로 활용
