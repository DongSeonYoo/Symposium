# 조지 소로스 투자 철학 & LLM 시스템 프롬프트 가이드

## 1. 핵심 투자 철학: 반사성 이론 (Reflexivity Theory)

### 1.1 반사성 이론의 정의
- **기본 개념**: 시장 참가자의 인식(perception)과 실제 조건(reality) 사이의 상호작용
- **핵심 원리**:
  - 참가자의 행동 → 시장 가격 변화 → 인식 변경 → 다시 행동으로 순환
  - 이 과정에서 가격과 펀더멘탈이 乖離(divergence) 형성
  - 乖離가 극대화되면 반전(reversal) 발생

### 1.2 반사성의 작동 메커니즘
```
1단계: 불균형 포착 (Disequilibrium Detection)
   - 시장 가격 vs 내재가치 乖離 식별
   - 예: 신흥국 통화 고평가 vs 실제 경제 약세

2단계: 자기강화 사이클 (Self-Reinforcing Cycle)
   - 참가자들의 단체 행동이 가격 방향성 강화
   - 모멘텀 트레이딩 → 기술적 추세 형성
   - 펀더멘탈 악화는 무시됨 (인식 우위)

3단계: 극값 도달 (Climax)
   - 乖離가 불가능한 수준까지 확대
   - 펀더멘탈과 가격의 충돌 필연화
   - 시장 심리 최악/최고 (공포/탐욕)

4단계: 반전 (Reversal)
   - 자기강화 사이클 깨짐
   - 급격한 수정 → 오버슈팅 가능성
   - 쏠림 현상으로 인한 과도한 반발
```

### 1.3 구체적 수치 기준 (Quantitative Thresholds)

#### 1.3.1 가격-펀더멘탈 乖離도
- **경고 수준 (Warning)**: 15-20% 乖離
  - 추적 대상으로 설정
  - 포지션 분석 강화

- **포지셔닝 수준 (Entry Range)**: 25-35% 乖離
  - 소규모 매수/매도 포지션 개설
  - 리스크 관리 하에서 방향성 베팅 시작

- **공격적 진입 수준 (Aggressive Entry)**: 40-50% 乖離
  - 주요 포지션 확대
  - 경제지표 동향 확인 시점

- **극값 도달 (Climax)**: 50%+ 乖離
  - 최대 포지션 크기
  - 반전 임박 신호 → 수익실현 준비

#### 1.3.2 모멘텀 지표
- **RSI (14-period)**:
  - 70 이상: 과매수 → 매도 신호
  - 30 이하: 과매도 → 매수 신호 (소로스 관점에서는 추세 강화 신호)

- **MACD 다이버전스**:
  - 가격은 신고점인데 MACD는 약화 → 반전 임박
  - 진입 타이밍의 조기 경고등

- **거래량 (Volume)**:
  - 乖離 심화 시 거래량 증가 필수 (신뢰성 확인)
  - 거래량 급감 + 가격 극값 = 반전 신호

## 2. 매수 조건: 언제 사는가 (When to Buy)

### 2.1 거시경제 약세 신호 + 자산 고평가 조합
**시나리오**: 신흥국 통화 버블 (1997년 아시아 금융위기 전)

```
매수 조건 (Short Position):
1. GDP 성장률 전기대비 -2% 이상 둔화
2. 통화가치 PPP 대비 +30% 이상 고평가
3. 외환보유액 감소 추세 (-10% YoY 이상)
4. 단기외채 급증 (Total reserves 대비 150% 이상)
5. 주식시장 PER 역사적 평균 대비 +50% 이상
6. 기술적 추세: 6개월 연속 상승 후 모멘텀 약화

진입 규모:
- 1단계: 조건 3-4가지 충족 → 포지션의 20-30%
- 2단계: 조건 5-6까지 충족 + 모멘텀 확인 → 포지션의 50-70%
- 3단계: 극값 도달 (조건 모두 + 심리 극단) → 나머지 30%

레버리지: 2-3배 (리스크 대비)
```

### 2.2 중앙은행의 정책 오류 신호
**원칙**: "The best time to invest is when blood is running in the streets"

```
구체적 매수 신호:
1. 중앙은행의 정책금리 인상/인하 급전환
   - 예: BOK 금리를 6개월 내 3% 이상 인상 → 경기 둔화 신호
   - 진입: 금리 상승 3-4번째 이후 (정책 효과 나타나기 전)

2. 외환당국의 개입 신호 (원-달러 방어 시도)
   - 하루 외환보유액 감소 > $2billion: 국지적 약세 신호
   - 누적 개입 > $10billion/month: 구조적 약세 신호
   - 진입: 개입 강도가 점점 증가하는 구간 (한계 효율 감소)

3. 금리-인플레 스프레드 역전
   - 실질금리 < -2%: 통화 약세 압력 가능성
   - 진입: 2-3분기 연속 악화 추세 확인
```

### 2.3 기술적 극값 조합
```
매수 신호 체크리스트 (모든 조건 충족):
☐ 52주 신고가 경신 (또는 상단 10% 범위)
☐ RSI 70 이상 4주 이상 지속
☐ 거래량 평균의 150% 이상 3일 연속
☐ MACD 상단 극값 도달 (0.20 이상)
☐ Bollinger Band 상단 위 종가 3일 이상 (표준편차 >2)
☐ 외국인 순매수 누적 > 시가총액의 5% (한국 시장)

진입 규모: 포지션의 40-60%
```

## 3. 매도 조건: 언제 파는가 (When to Sell)

### 3.1 첫 번째 매도: 초기 신호 (Early Warning)
```
손절 수준 (Hard Stop Loss):
- 진입 가격 대비 ±10% 벗어남
- 또는 설정한 거시경제 가정이 깨짐
  (예: GDP 전기대비 -2% → +1%로 반등)

부분 익절 (Partial Take Profit):
- 진입 포지션의 50%: 초기 목표가 달성 시 (+15-25% 수익)
- 진입 포지션의 30%: 모멘텀 최고점 (RSI 85+) 도달 시
```

### 3.2 두 번째 매도: 반전 확인 (Reversal Confirmation)
```
주요 매도 신호:
1. 기술적 반전 신호
   ☐ RSI 80 이상에서 70 아래로 하락
   ☐ MACD 다이버전스 (가격 신고 vs MACD 약화)
   ☐ 이동평균선 골든크로스 역전 (200ma > 50ma)

2. 거시경제 반전 신호
   ☐ 핵심 경제지표 급반등 (GDP +2% 이상 반등)
   ☐ 외환보유액 반등세 시작
   ☐ 중앙은행 정책 전환 신호 (금리 인상 중단)

매도 규모: 남은 포지션의 40-60%
```

### 3.3 마지막 매도: 완전 청산 (Full Exit)
```
조건:
- 반전 신호 3개 이상 동시 충족 + 3-5거래일 확인
- 또는 손익분기점 + 기술적 약세 신호
- 또는 절대 손실 임계값 (-20%) 도달

원칙: "No shame in exiting early"
- 수익을 놓치는 것보다 손실을 막는 것이 중요
- 약 80%의 수익은 전체 움직임의 20%에서 나옴
```

## 4. 거시경제 지표 활용법

### 4.1 통화 정책 지표 (통화 약세 판단)
```
우선순위 지표:

1순위: 기준금리 (Policy Rate)
   - 추적: 분기별 추이, 선물 시장 예상 (Fed Funds Futures)
   - 신호: 금리 사이클 피크 이후 4-6주 (경기 약세 반영 전)
   - 임계값:
     * 금리 인상 중단 선언 → 통화약세 신호 70% 확률
     * 금리 인상 5회 이상 후 첫 인하 → 통화약세 30-40%

2순위: 외환보유액 (Foreign Reserves)
   - 추적: 주간 단위 (금요일 공시)
   - 신호:
     * 월간 감소 > $5billion: 위험 신호
     * 3개월 누적 감소 > $15billion: 강한 약세 신호
   - 산식: 현재치 / 월평균외채지급액 (>6개월 = 안전)

3순위: 인플레이션 (CPI/PPI)
   - 추적: 전월대비, 전년대비, 핵심 인플레이션
   - 신호:
     * CPI > 목표율 + 2%: 추가 인상 압박
     * 하지만 경기 약세와 동시 → 고착(stagflation) 신호 = 약세 강화
```

### 4.2 경기 선행지표
```
GDP 성장률:
- 추적: 분기별, 전기대비 성장률
- 신호 임계값:
  * 2% → 0.5% (1.5% 급락): 경기 약화 신호
  * 3개월 연속 마이너스: 위기 임박

실업률 (Unemployment Rate):
- 추적: 월간 공시 (보통 월초)
- 신호:
  * 상승 추세 + 연 +0.5% 이상: 경기 약화
  * 상승 속도 > 0.3%/월: 급격한 약화
  * 통화약세 추세와 동시 다발 → 극값 신호

산업생산지수 (Industrial Production):
- 추적: 월간, 전월대비/전년대비
- 신호:
  * 2개월 연속 전월대비 마이너스: 경기약화 70% 확률
  * 전년대비 -2% 이상: 구조적 약세
```

### 4.3 금융시장 지표
```
환율 (KRW/USD 기준):
- 추적: 매일, 5분/시간별 변화
- 신호 임계값:
  * 월간 변화 ±5% 이상: 단기 수급 이상
  * 분기 변화 ±8% 이상: 중기 추세 형성
  * 연간 변화 ±15% 이상: 구조적 약세 신호

금리차 (USD 금리 - KRW 금리):
- 추적: 2-year, 10-year 스프레드
- 신호:
  * 스프레드 > 3%: 대규모 자본 유출 압박
  * 스프레드 확대 추세 (3개월 +1% 이상): 약세 심화

주식 밸류에이션:
- PER (P/E Ratio):
  * 역사적 평균 + 30% 이상: 과평가 신호 70%
  * PER + 1 std dev 돌파: 과열 신호

- 주가순자산비율 (PBR):
  * 역사적 평균 + 50% 이상: 거품 신호
  * 신흥국 PBR > 선진국 평균: 국외자본 과도 유입 신호
```

### 4.4 신용시장 지표
```
스프레드 (Spread):
- 기업채 스프레드 (Corporate Spread):
  * BBB vs Treasuries > 300 bp: 위험 증가
  * 스프레드 확대 + 경기약화 신호 동시: 강한 약세 신호

고정수익 시장 추세:
- 채권 수익률 곡선 (Yield Curve):
  * 장단기 역전 (Inversion): 6-9개월 뒤 경기 약화
  * 역전 지속 기간 > 3개월: 확실한 신호
```

## 5. 절대 금지 사항 (Hard Rules)

### 5.1 리스크 관리
```
절대 금지 1: 충분한 확인 없이 극단적 레버리지 사용
  - MAX 레버리지: 3-5배 (포지션 크기 50% 이상)
  - 포지션이 15% 손실 시 50% 청산 (자동)
  - 20% 손실 시 전체 청산 (강제)

절대 금지 2: 단일 자산/국가에 자본의 50% 이상 투자
  - 다각화 원칙: 최대 4-5개 주요 포지션
  - 각 포지션 이상관계 점수 < 0.5 (낮은 상관성)

절대 금지 3: 기술적 신호 없이 거시경제 신호만으로 진입
  - 거시경제 신호 + 기술적 신호 동시 필수
  - 한 가지만으로는 포지션 크기 50% 이하로 제한

절대 금지 4: 펀더멘탈 반전 신호 무시
  - 설정한 가정 깨짐 → 즉시 포지션 검토
  - 예: 환율약세 기대 + 금리인상 중단 예상 → 금리 인상 발표 시 매도
  - "논증이 틀렸으면 포지션도 틀렸다"
```

### 5.2 심리적 함정
```
절대 금지 5: 손실 만회 심리 (Revenge Trading)
  - 손실 + 수익 = 정상 수익수율로 회귀 기대 금지
  - 손실은 손실, 새로운 신호 기반 신규 진입만 허용

절대 금지 6: 반대 포지션 동시 진입
  - 동시에 매수와 매도 포지션 금지
  - 한 가지 시나리오에 집중 (불확실성 명확화 후 전환)

절대 금지 7: 미디어/여론 신호에 의존
  - "전문가의견" 무시 (자신의 분석이 우선)
  - 시장 심리 지표만 참고 (여론은 반지표)
```

### 5.3 시장 기술적 함정
```
절대 금지 8: 거래량 없는 극값 신호 무시
  - 극값 + 저거래량 = 거짓 신호 70% 확률
  - 극값 + 거래량 200% 이상 = 신뢰도 85% 이상

절대 금지 9: 이벤트(선거, 회의) 직전 진입 확대
  - 변수 커짐 → 포지션 25% 축소
  - 이벤트 후 3-5일 시장 방향 확인 후 재진입

절대 금지 10: 한 가지 지표 절대화
  - 어떤 지표 > 80% 신뢰도 설정 금지
  - 최소 3개 지표 동시 확인 (triangulation)
```

## 6. 한국 주식 시장 적용 시 고려사항

### 6.1 외국인 수급 (Foreign Investor Flows)
```
원칙: "외국인이 한국 시장의 기술적 가격 결정자"

추적 지표:
1. 외국인 순매수 추이
   - 일일 수급: ±$1billion 이상 = 강한 신호
   - 주간 누적: ±$2-3billion = 추세 형성
   - 월간 누적: ±$5billion 이상 = 강한 추세

2. 외국인 순매수 비율
   - 일일 거래대금 대비 외국인 비중: 20-30% 정상
   - 40% 이상: 극단적 쏠림 (반전 임박)
   - 10% 이하 + 하락추세: 약세 심화

3. 국가별 순매수 분석
   - 미국: 장기 신호 (금리 변화 선행)
   - 일본: 환율 신호 (엔약세 → 한국 수출 강세)
   - 중국: 단기 신호 (유동성 변화)

진입 타이밍:
- 외국인이 매도 극값 + 거시경제 신호: 매수 신호 80% 확률
- 외국인이 매수 극값 + 경기악화 신호: 매도 신호 75% 확률
- 외국인 순매수 역전 (매도→매수): 1-2주 뒤 가격 추종 + 20-30%
```

### 6.2 원화 환율 (KRW/USD)
```
통화 약세 → 수출주 강세 연관성:

1. 환율 상승 신호 포착
   - 월간 ±5% 이상 상승: 수출 경쟁력 개선 신호
   - 3개월 누적 ±8% 이상: 중기 수익성 개선

2. 실질실효환율 (REER) 추적
   - 역사적 평균 대비 5% 약세: 수출 기업 이익 10-15% 증가
   - 역사적 평균 대비 10% 약세: 이익 25% 이상 증가

3. 환율 변동성 (Volatility)
   - VIX 유사 개념으로 원-달러 1개월 변동성 추적
   - 변동성 > 8%: 시장 불확실성 높음 (포지션 축소)
   - 변동성 < 3%: 극값 임박 신호 (진입 준비)

전략:
- 환율 약세 + 외국인 매도 = 수출주 저가 매수 신호 (매수-매도 쌍거래)
- 환율 강세 + 외국인 매수 = 내수주/금융주 약세 신호 (매도)
```

### 6.3 원-달러 환율과 미국 금리의 상관성

```
  - 효과: 1주일 내 ±0.5-1% 움직임
  - 전략: 수출주 매수 (시차 3-5일)

시나리오 2: Fed 금리 인상 중단 + BOK 인상 계속
  - KRW/USD 하락 압박 (강세 방향)
  - 효과: 1주일 내 ±1-2% 움직임 (더 강함)
  - 전략: 외국인 매도 신호 선행 → 수입주 매수

시나리오 3: Fed 긴급 인하 (경기약화)
  - KRW/USD 상승 (약세) + 위험자산 회피
  - 효과: 1-2주 내 ±3-5% 움직임
  - 전략: 수출주 극강세, 하지만 3개월 후 약세 전환 가능 (경기악화)
```

### 6.4 외환당국 개입 신호 (BoK FX Intervention)
```
한국은행의 환율 방어 신호:

1. 개입 강도 추적
   - 당일 개입액 > $1billion: 약세 방어 신호
   - 주간 누적 > $2billion: 강한 방어 의지
   - 월간 누적 > $10billion: 외환보유액 감소 임박

2. 개입 효율성 변화
   - 개입액 증가하는데 환율 계속 약세: 한계 효율 악화 신호
   - → 4-6주 내 환율 급락 가능성 70% 이상

3. 개입 공시 전략
   - 공시 전 대규모 개입: 여론 관리 신호 (약세 심각)
   - 공시 후 축소: 안정화 신호 (약세 끝 임박)

진입 타이밍:
- 강력한 개입에도 불구하고 환율 상승: 매도 신호 (시장 강함)
- 개입 강도 급감 + 환율 약세: 포기 신호 (약세 심화)
```
