# Symposium — 투자 판단 Quick Reference

> LLM 페르소나 시스템에 실전 투자 분석을 적용하기 위한 체크리스트

## 1. 분석 시작 전 수집 데이터 체크리스트

### 기업 재무 데이터 (필수)
```
□ 최근 4분기 EPS (earnings per share)
□ 최근 4분기 매출액
□ 최근 4분기 순이익
□ PER, PBR, ROE, ROA
□ 부채비율 (D/E)
□ 자유현금흐름 (FCF)
□ 순이익률 추세
□ 시장점유율 (가능하면)
```

### 거시경제 지표 (분석 당시)
```
□ VIX (공포지수)
□ 미국채 10년물 수익률
□ 달러 인덱스 (DXY)
□ 원달러 환율
□ WTI 유가
□ 금 가격
□ KOSPI 일일 등락률
□ Fed 최근 발언 요약
□ 한국은행 최근 발언 요약
```

### 수급 데이터 (한국 시장)
```
□ 외국인 3개월 누적 매수/매도액
□ 기관 주간 매매 현황
□ KOSPI/KOSDAQ 비율
□ 해당 종목 외국인 비중
□ 해당 종목 기관 비중
```

## 2. 페르소나별 핵심 질문

### Warren Buffett 체크
```
Q1. 이 회사의 ROE가 15% 이상인가?
    → NO: 매수 회피
    → YES: Q2로

Q2. PBR이 1.5 이하인가? (또는 PBR이 낮고 ROE가 높은가?)
    → NO: 신중 (고평가 가능성)
    → YES: Q3로

Q3. 부채비율(D/E)이 100% 이하인가?
    → NO: 위험 (회피)
    → YES: Q4로

Q4. 자유현금흐름이 순이익의 80% 이상인가?
    → NO: 주의
    → YES: Q5로

Q5. 당신이 10년 동안 이 회사 주식을 보유할 수 있는가?
    → NO: 회피
    → YES: 매수 신호 ✓

최종 점수: 위 5가지 모두 YES면 70점 이상
```

### George Soros 체크
```
Q1. 글로벌 거시 불균형이 보이는가?
    - 금리 급등하는데 신흥국 계속 상승? (이상함)
    - 달러 강세하는데 신흥국 자금 유입? (기회)
    - 기준금리 고금리인데 인플레 안 잡혀? (불균형)
    → YES면 기회, NO면 일반적 추세

Q2. 외국인 자금 흐름이 불균형을 반영하고 있는가?
    - 외인이 아직도 신흥국 매도 중? (미처 인식 못 함)
    - 외인이 준비 없이 매도 시작? (과도한 공포)
    → YES면 반사성 기회

Q3. 다음 3~6개월간 인식이 바뀔 가능성은?
    - Fed가 금리 인상 멈춘다고 신호? (인식 변화 예상)
    - 경기 데이터 좋아지고 있나? (회복 신호 확산)
    → YES면 매매 신호

최종 판단: 불균형 + 자금 미반영 + 인식 변화 임박 = BUY/SELL 신호
```

### Ray Dalio 체크
```
Q1. 현재 경제 사이클이 어느 단계인가?
    [ ] 성장 + 저인플레 (주식 강세)
    [ ] 성장 + 고인플레 (금·원자재 강세)
    [ ] 약성장 + 고인플레 (방어 필요)
    [ ] 약성장 + 저인플레 (채권 강세)

Q2. 포트폴리오 리스크 노출이 환경에 맞는가?
    - 만약 금리가 +100bps 상승하면? (손실 X%)
    - 만약 인플레가 +1pp 상승하면? (손실/수익 X%)
    - 만약 경기가 마이너스 성장하면? (손실 X%)
    → 각 시나리오 손실 < 5%?

Q3. 리밸런싱이 필요한가?
    - 주식 비중이 목표보다 5% 이상 높은가?
    - 채권 비중이 부족한가?
    - 현금 비중이 충분한가?
    → YES면 리밸런싱

최종 판단: 위험 균형이 맞으면 포지션 유지, 맞지 않으면 조정
```

### Peter Lynch 체크
```
Q1. 5가지 조건을 모두 충족하는가?
    [ ] 당신이 이해하는 비즈니스?
    [ ] 업종이 성장 중? (CAGR ≥ 10%)
    [ ] 회사가 리더? (시장점유율 Top 3)
    [ ] 매출/이익 성장 가속? (≥ 20%, 가속도)
    [ ] PEG < 1.0? (또는 0.8 < PEG < 1.2 + ROE > 20%)

모두 YES면 STRONG BUY
3개 이상 YES면 BUY
2개 이하 YES면 AVOID

Q2. 성장 모멘텀이 유지되고 있는가?
    분기별 성장률 추이:
    Q1: X% → Q2: X% → Q3: X%
    상승 추세? → YES 유지, NO 주의신호

Q3. PEG 재상승은 없는가?
    - PEG 1.5 이상 올라가면 수익 일부 실현

최종 판단: 5가지 + 성장 가속 + PEG 양호면 매수, 하나라도 깨지면 재검토
```

### Hyunjoo Park 체크
```
Q1. 외국인 자금 흐름은?
    □ 3개월 누적 매수 > 5조원 + 비중 증가 = 강한 상승신호
    □ 3개월 누적 매도 > 5조원 = 약세 신호
    □ 월간 순매도액 > 3조원 지속 = 회피 필요

Q2. 원달러 환율은?
    □ 1,100~1,200원 = 중립
    □ 1,200원+ = 수출주 강세 신호 (반도체, 화학, 자동차)
    □ 1,100원 이하 = 금융주 강세 신호

Q3. 정부 정책이 지원하는 산업인가?
    □ 최근 정부 발표 또는 계획에서 해당 산업이 포함되었나?
    □ R&D 예산이 증가 중인가?
    → YES면 추가 가점

Q4. KOSPI/KOSDAQ 비율은?
    □ > 2.0 = 대형주(반도체, 금융) 선호 시대
    □ < 1.7 = 중소형주(바이오, 신성장) 선호 시대
    → 섹터 선택의 나침반

최종 판단: 외인 매수 + 정책 지원 + 환율 유리 = 3박자 완성
```

## 3. 종목별 매수 신호 체크리스트

### Buffett 타입 (가치주) — 종합점수 70점 이상
```
점수 배분:
- PBR/밸류에이션: 20점
- ROE 지속성: 20점
- 경쟁 해자: 20점
- 재무 건전성: 20점
- 경영진 신뢰도: 20점

목표 보유 기간: 5~10년
목표 수익률: 연 10~15%
리스크: 낮음

예시 종목:
- 삼성전자 (경쟁력, 배당)
- 현대차 (저PER, 배당)
- 삼성생명 (배당, 자산가치)
```

### Lynch 타입 (성장주) — PEG < 1.0 + 성장 가속
```
체크:
- PEG: 0.8 이하 (필수)
- 분기 EPS 성장: 25% 이상 + 가속 추세
- ROE: 15% 이상 + 개선 중
- 시장점유율: 확대 추세

목표 보유 기간: 2~5년
목표 수익률: 50~200%
리스크: 중간~높음

예시 종목:
- 반도체 신진 회사 (성장주)
- 신약 개발 초기 제약사 (성장 임박)
- 신산업 선도사 (배터리, 수소)
```

### Soros 타입 (거시 베팅) — 글로벌 불균형 포착
```
신호:
- 시장의 광범위한 오판이 있는가?
- 외인 자금이 이를 반영하지 않는가?
- 향후 3~6개월 인식 변화 가능성?

목표 보유 기간: 3~6개월
목표 수익률: 20~50%
리스크: 높음 (타이밍 민감)

예시 거래:
- 원달러 환율 급등 시 수출주 매수
- 외인 집단 매도 시 저가 매수
- 금리 급락 시 주식 매수
```

### Dalio 타입 (올웨더) — 리스크 균형
```
체크:
- 포트폴리오 손실 시나리오 < 5%?
- 각 자산군 비중이 균형인가?
- 리밸런싱 필요한가?

목표 보유 기간: 장기
목표 수익률: 연 6~8%
리스크: 낮음 (분산으로 보호)

포트폴리오 배분:
- 주식 30% (경기 위험)
- 채권 40% (금리 위험)
- 금 15% (인플레 위험)
- 원자재 10% (공급 위험)
- 현금 5% (유동성)
```

### Park 타입 (한국 시장) — 수급 + 정책
```
신호:
- 외인 3개월 매수 > 5조원?
- 정부 정책이 지원하는 산업?
- 환율이 수출주에 유리?
- KOSPI/KOSDAQ 비율 적정?

목표 보유 기간: 3~6개월 (단기) ~ 2~3년 (중기)
목표 수익률: 20~80%
리스크: 중간 (수급 변화에 민감)

예시 종목:
- 반도체 (외인 매수 + 환율 유리)
- 배터리 (정책 지원 + 성장)
- 금융주 (금리 인상기 수익성 개선)
```

## 4. 매도 신호 체크리스트

### 공통 매도 신호 (모든 페르소나)
```
Level 1 (즉시 매도):
□ 기업 공시에서 위험 신호 (적자 전환, 대규모 손실)
□ CEO/CFO 긴급 사퇴
□ 주력 사업 폐지 공고
□ 대규모 부도 우려 뉴스
```

### Buffett 매도 신호
```
Level 2 (재평가 필요):
□ ROE 분기 연속 하락 (3분기+)
□ FCF 음수 전환
□ 배당금 감액 발표
□ 부채비율 100% 급상승
```

### Lynch 매도 신호
```
Level 2 (성장 모멘텀 상실):
□ EPS 성장률 < 15%로 둔화
□ 3분기 EPS 성장 마이너스
□ 컨센서스 3회 연속 하향
□ PEG > 1.5 재상승 (성장률 유지)
□ 원래 투자 논거 무너짐 (신약 실패, 제품 부진 등)
```

### Soros 매도 신호
```
Level 2 (불균형 해소 또는 반전):
□ 외인 매도로 전환된 후 1개월 지속
□ VIX 급상승 후 정상화 시작 (공포 극복)
□ 거시 신호 반전 (금리 인상 → 인상 멈춤)
□ 글로벌 자금 흐름 역전
```

### Dalio 매도 신호
```
Level 2 (리스크 예산 도달):
□ 포트폴리오 손실이 최대 허용선의 60% 도달
□ 경제 사이클 환경 변화 신호 (금율 급등, 인플레 급등)
□ 신용 스프레드 > 250bps 확대
□ 리밸런싱 필수 (주식 50% 이상으로 증가)
```

### Park 매도 신호
```
Level 2 (수급 악화):
□ 외인 월간 순매도 > 3조원 지속
□ 원달러 환율 급등 (1,250원+) 또는 급락 (1,050원-)
□ 정부 정책 변화 (지원 산업 제외)
□ KOSPI/KOSDAQ 비율 급변 (섹터 회전 신호)
□ VIX > 25 + 외인 매도 동조
```

## 5. 실전 의사결정 플로우

### 신규 종목 추가 시
```
Step 1. 데이터 수집 (필수 지표 체크)
        → 데이터 불완전하면 분석 미연기

Step 2. 5개 페르소나 개별 평가 (동시 진행)
        - Buffett: 가치 평가
        - Lynch: 성장 평가
        - Soros: 거시 신호
        - Dalio: 리스크 평가
        - Park: 수급 신호

Step 3. 각 페르소나 신뢰도 가중치 적용
        (최근 3개월 적중률 기반)

Step 4. 가중합산 (Opus 4.5)
        → 최종 점수 계산

Step 5. 포지션 결정
        - 점수 > 70: STRONG BUY (비중 증가)
        - 점수 50~70: BUY (신규 진입 또는 비중 유지)
        - 점수 30~50: HOLD (보유 또는 대기)
        - 점수 < 30: SELL (매도)

Step 6. 리스크 체크
        - 최대 손실 시나리오: X%
        - 포트폴리오 영향: Y%
        → 통과하면 진입 승인
```

### 보유 종목 정기 검토 (주 1회)
```
Step 1. 주간 실적/뉴스 확인
        □ 부정적 뉴스?
        □ 실적 변화?
        □ 경영진 변화?

Step 2. 주간 기술 지표 체크
        □ 매도 신호 발생?
        □ 수급 악화?
        □ 거래량 감소?

Step 3. 필요시 재평가
        매도 신호 3개 이상 → 매도 검토
        매도 신호 1~2개 → 대기

Step 4. 포트폴리오 리밸런싱
        비중 5% 이상 차이 → 조정
```

### 매월 전략 회의 (월 1회)
```
Agenda:
1. 마지막 달 실적 검토
   - 각 페르소나별 적중률
   - 가중치 자동 조정

2. 거시경제 전망 업데이트
   - VIX, 금리, 환율 추세
   - 다음 달 예상 환경

3. 섹터 회전 신호
   - KOSPI/KOSDAQ 비율 추세
   - 정부 정책 변화

4. 포트폴리오 재구성
   - 비중 조정
   - 신규/기존 종목 검토

5. 위험 평가
   - 최대 손실 시나리오
   - 현금 비중 조정
```

## 6. 페르소나 신뢰도 자동 조정

매주 실적 공시 또는 중요 뉴스 후:

```
각 페르소나의 판단을 "실제 결과"와 비교:

예: 페르소나가 "강한 BUY" → 실제 종목 1개월 -10% 하락
    - Buffett의 신뢰도: -5점 (재무 판단 외)
    - Lynch의 신뢰도: -10점 (성장 예측 실패)

역으로: 페르소나가 "SELL" → 실제 종목 1개월 +20% 상승
        - Soros의 신뢰도: -5점 (거시 판단 외)
        - Park의 신뢰도: -10점 (수급 예측 실패)

누적 스코어:
- 시작 기본값: 각 페르소나 1.0 (100%)
- 매주 ±0.05~±0.10 조정
- 최소 0.5 (50% 가중치), 최대 1.5 (150% 가중치)

3개월 마다 리셋 가능 (시장 변화 반영)
```

## 7. 비상 상황 Protocol

### 위기 모드 진입 신호
```
다음 조건 중 2개 이상 동시 발생:

□ VIX > 30
□ 원달러 환율 > 1,250원 (1주일 연속)
□ 외인 월간 순매도 > 5조원
□ S&P 500 -10% 이상 (약세장)
□ 미국 금리 급등 (> 4.5%)

→ 위기 모드 ACTIVATE
```

### 위기 모드 포트폴리오 (모든 페르소나 동의)
```
현금: 40~50%
```

## 9. 용어 정리

| 용어 | 의미 | 기준 |
