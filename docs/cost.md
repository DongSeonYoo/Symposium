## 1회 분석 사이클 비용 추정

페르소나 5개 × 3라운드 기준:

| 호출 | 모델 | Input 토큰 | Output 토큰 |
|------|------|-----------|------------|
| Round 1 독립판단 × 5 | Sonnet 4.6 | 15,000 | 5,000 |
| Round 2 교차비판 × 5 | Sonnet 4.6 | 25,000 | 5,000 |
| Round 3 재반론 × 5 | Sonnet 4.6 | 30,000 | 4,000 |
| 최종 합산 × 1 | **Opus 4.5** | 8,000 | 500 |
| **합계** | | **~78,000** | **~14,500** |

```
Sonnet 4.6: (70,000 × $3 + 14,000 × $15) / 1,000,000 = $0.42
Opus 4.5:   (8,000  × $5 + 500   × $25) / 1,000,000 = $0.05
1회 합계: 약 $0.47
```

## 비용 최적화 전략

1. **모델 티어 분리**: Round 1~3은 Sonnet 4.6, 최종 합산만 Opus 4.5
2. **프롬프트 캐싱 필수**: 페르소나 시스템 프롬프트 캐시 (수천 토큰 반복)
3. **감시 종목 수 제한**: 10개 이하 유지 시 비용 선형 제어 가능
4. **Batch API 활용**: 히스토리 분석 등 비긴급 작업은 50% 할인 배치 처리
