## 거시경제 신호: 우선순위 지표 (Top 5)

### 1순위: 기준금리 추이 (Policy Rate Trend)
```
신호: 금리 인상 중단 또는 인하 예상
신뢰도: 70-85%
시차: 신호 포착 후 4-6주
예시: BOK 인상 사이클 종료 → 통화약세 신호
```

### 2순위: 외환보유액 (Foreign Reserves)
```
신호: 월간 감소 > $5B 또는 누적 > $15B
신뢰도: 80-90%
시차: 신호 포착 후 2-3주
예시: 외환보유액 급감 → 환율 약세 압박
```

### 3순위: GDP 성장률 (Economic Growth)
```
신호: 전기대비 -1% 이상 악화 또는 3개월 연속 마이너스
신뢰도: 75-85%
시차: 신호 포착 후 2-4주
예시: GDP 0.5% → -0.5% (1% 급락) = 경기약화 신호
```

### 4순위: 실업률 & 산업생산지수 (Labor & Production)
```
신호: 실업률 상승 + 산업생산 2개월 연속 마이너스
신뢰도: 70-80%
시차: 신호 포착 후 3-5주
예시: 동시 악화 = 경기 급락 신호 (신뢰도 85%)
```

### 5순위: 신용스프레드 (Credit Spreads)
```
신호: 기업채 스프레드 > 300bp 또는 급확대
신뢰도: 75%
시차: 신호 포착 후 1-2주
예시: 스프레드 확대 + 경기약화 신호 동시 = 신용위기 신호
```

## 기술적 신호: 우선순위 지표 (Top 5)

| 순위 | 지표 | 신호 | 신뢰도 |
|------|------|------|--------|
| 1 | RSI (14) | 70 이상 4주+ 또는 30 이하 4주+ | 85% |
| 2 | 거래량 | 평균 150%+ 3일 연속 + 가격 극값 | 85% |
| 3 | MACD | 다이버전스 (가격 신고 vs MACD 약화) | 80% |
| 4 | Bollinger Band | 상단/하단 이탈 (표준편차 >2) | 75% |
| 5 | 이동평균 | 50ma > 200ma 역전 (골든크로스 해체) | 75% |

### 기술적 신호 조합 신뢰도
```
1개 신호만: 신뢰도 30-40% (사용 금지)
2개 신호: 신뢰도 50-60% (포지션 최소)
3개 신호: 신뢰도 75-80% (정상 진입)
4개 신호: 신뢰도 85-90% (적극적 진입)
5개 신호: 신뢰도 95%+ (극값 신호)
```

## 한국 시장 고유 신호: 3가지 핵심

### 신호 1: 외국인 순매수 (Foreign Investor Net Buying)
```
극강 신호 (±$1B 일일):
  → 신뢰도 75% (기술적 가격 결정자)
  → 추세 반전 신호 1-2주 뒤 가격 추종 20-30%

강한 신호 (±$500M-1B):
  → 신뢰도 65% (거시경제와 조합 시 80%)

약한 신호 (±$100-500M):
  → 신뢰도 40% (모니터링용)

역전 신호 (매도→매수):
  → 매도 극값에서 매수로 역전 = 1-2주 내 반등 70% 확률
  → 매수 극값에서 매도로 역전 = 1-2주 내 하락 75% 확률
```

### 신호 2: 원-달러 환율 (KRW/USD)
```
월간 ±5% 이상: 중기 추세 신호
  → 약세(rate ↑): 수출주 강세, 기업 이익 +10-15%
  → 강세(rate ↓): 내수주 강세, 수출주 약세

분기 ±8% 이상: 강한 중기 신호
  → 약세: 반도체 업황 개선, 이익 +25% 이상
  → 강세: 수출 경쟁력 악화

실질실효환율 (REER):
  → 역사적 평균 대비 -5%: 수출 경쟁력 +10-15%
  → 역사적 평균 대비 -10%: 극값 신호, 수출주 극강세
```

### 신호 3: 한국은행 개입 (BoK FX Intervention)
```
소규모 개입 ($500M-1B): 정상적 시장 관리
중규모 개입 ($1-3B): 한은 본격 방어, 구조적 약세 신호
대규모 개입 ($3B+): 극급 방어, 외환보유액 감소 임박

개입 효율 악화:
  개입액 증가 + 환율 약세 지속
  → 한계 효율 악화 신호
  → 4-6주 내 환율 급락 70% 확률

개입 강도 급감:
  대규모 개입에서 축소로 전환
  → 한은 항복 신호
  → 약세 심화 신호
```

## 실전 진입 프로세스 (5단계)

### Step 1: 거시경제 신호 수집 (1주)
```
체크리스트:
□ GDP 성장률 최신 발표
□ 외환보유액 주간 추이
□ 기준금리 전망 (중앙은행 회의 예정)
□ 실업률 / 산업생산 월간 발표
□ 신용스프레드 (회사채 추이)

최소 요구: 4/5 신호 동일 방향
→ Step 2로 진행
```

### Step 2: 기술적 신호 확인 (3-5일)
```
체크리스트:
□ RSI 70 이상 또는 30 이하 (2주 이상 지속)
□ 거래량 150%+ (3일 연속) + 가격 극값
□ MACD 다이버전스 확인
□ 이동평균 역전 (50ma vs 200ma)
□ Bollinger Band 이탈

최소 요구: 3/5 신호 동일 방향
→ Step 3으로 진행
```

### Step 3: 한국 시장 신호 추가 확인 (즉시)
```
체크리스트:
□ 외국인 순매수 추이 (최근 1주)
□ 원화 환율 움직임 (최근 1개월)
□ 한은 개입 강도 변화 (최근 1주)
□ VKOSPI 레벨 (현재)

권장: 1개 이상 신호 충족
→ Step 4로 진행
```

### Step 4: 심리적 신호 확인 (즉시)
```
체크리스트:
□ 언론 쏠림 정도 (한 방향 보도 강함?)
□ 기관 포지셔닝 (극단적인가?)
□ VIX/VKOSPI 극값 (20+?)
□ 시장 심리 (공포 또는 탐욕?)

권장: 2/3 신호 충족
→ Step 5로 진행
```

### Step 5: 포지셔닝 & 진입
```
조건 확인:
거시 4-5/5 ✓ → 기술 3-5/5 ✓ → 심리 2-3/3 ✓

진입 규모:
- 초기 진입: 자본 20-30%
- 레버리지: 1-2배
- Target: 거시 신호 극값 도달 시 확대

손절 설정:
- Hard Stop: ±10% 또는 테시스 파괴
- 부분 익절: +15-25% 또는 RSI 85+
- 최종 청산: 반전 신호 3개 + 3-5일 확인
```

## 실패하는 트레이더 vs 성공하는 트레이더

### 실패 패턴 (7가지)

| 패턴 | 특징 | 결과 |
|------|------|------|
| 신호 무시 | 거시 신호 무시하고 기술만 보기 | 신뢰도 30-40%, 수익률 음수 |
| 집중도 위반 | 단일 포지션 > 50% | 극단적 손실 경험 (30-50%) |
| 감정 거래 | 손실 만회 (Revenge) 또는 공포 | 손실 +20-30% 추가 |
| 반대 포지션 | 동시에 롱/숏 진입 | 비용만 증가, 수익 0 |
| Hard Stop 무시 | -10% 손실을 -20-30%까지 유지 | 극단적 손실 |
| 미디어 신호 | "뉴스에서 강세라고 했으니" | 역전에 급습당함 (신뢰도 20%) |
| 과도한 레버리지 | 5배 이상 | 마진콜 또는 전량 손실 |

### 성공 패턴 (7가지)

| 패턴 | 특징 | 결과 |
|------|------|------|
| 신호 수렴 | 거시 4-5/5 + 기술 3-5/5 | 신뢰도 75-95%, 수익률 +15-50% |
| 다각화 | 4-5개 포지션, 각 25% 이하 | 변동성 낮음, 안정적 수익 |
| 자동 손절 | Hard Stop -10% 준수 | 손실 최소화, 생존 기간 연장 |
| 부분 익절 | 50% at +15-25%, 30% at RSI 85+ | 수익 확정, 심리 안정 |
| 신호 기반 | "이 거래가 내 원래 이유를 지지하는가?" | 틀린 거래 빨리 종료 |
| 시장 추종 | 시장 신호 기반 거래 (예측 배제) | 극값 전에 진입, 극값 후 퇴출 |
| 적절한 레버리지 | 초기 1-2배, 극값 3배 | 생존 기간 길고 수익 극대화 |


```
  - 기술적 데이터 (RSI, MACD, 거래량)

  - 한국 시장 특화 신호 (외국인, 환율, 한은)

  - 신호 수렴도 계산 (거시/기술/심리 5/5 체크)
  - 불균형 포착 (가격-펀더멘탈 乖離도)
  - 자기강화 사이클 감지 (모멘텀 확인)
  - 극값 평가 (반전 임박 신호)

출력 레이어:
  - 매수/매도 신호 (방향)
  - 포지션 크기 (20-100%)
  - 레버리지 (1-5배)

검증 레이어:
  - Hard Stop Loss -10% (자동 실행)
  - 부분 익절 +15-25% (자동 실행)
  - 테시스 파괴 (즉시 재검토)
```
